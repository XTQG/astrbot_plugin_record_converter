## 🤝 介绍

QQ语音转化插件，包括语音转文件，文件、文本转语音，克服了QQ语音无法转发的问题, 同时可充当tts来使用

## 📦 安装

在astrbot的插件市场搜索astrbot_plugin_record_converter，点击安装即可

## ⌨️ 配置

请前往插件配置面板进行配置

## 🤝 使用说明

### 命令表

| 命令  | 说明         |
| --- | ------------- |
| 智能语音 | 根据上下文判断是否需要转语音 |
| 转语音 | 把文本/文件转成语音 |
| 转文件 | 把语音消息保存为文件 |
| （纯文本消息） | 有概率自动转成语音来发送（即tts） |
